# Ahxan Battle - Official Download & Info App

Welcome to the official **Ahxan Battle** download and information platform! This is a modern, responsive web application designed to showcase the Ahxan Battle game and provide players with easy access to download the APK.

## 🎮 Features

- **Beautiful Hero Section**: Eye-catching landing page with animated logo
- **Game Features**: Showcase of 6 amazing game features with icons
- **Screenshots Gallery**: Dedicated section for game screenshots
- **Download Section**: Easy download buttons with system requirements
- **Player Reviews**: Testimonials from active players
- **Responsive Design**: Fully mobile-friendly interface
- **Modern UI**: Dark theme with cyan accents and smooth animations
- **Smooth Animations**: Particle effects and scroll animations

## 📱 System Requirements

- **Android**: 5.0 or higher
- **Storage**: 150MB minimum
- **Internet**: Connection required
- **Browser**: Any modern web browser for the website

## 🚀 Quick Start

1. Clone the repository:
```bash
git clone https://github.com/Ahxanstore/ahxan-battle-app.git
cd ahxan-battle-app
```

2. Open in your browser:
```bash
# Simply open index.html in your browser
open index.html
# or
start index.html
```

3. That's it! The app is ready to use.

## 🎨 Customization

### Change Logo
Replace `1000143019.jpg` with your app logo image.

### Update Download Link
Edit the download URL in `index.html`:
```html
<a href="YOUR_DOWNLOAD_LINK_HERE" ...>
```

### Modify Colors
Update the CSS variables in `style.css`:
```css
:root {
    --primary-color: #007BFF;
    --secondary-color: #0056b3;
    --background: #0A0A0A;
    --text-color: #FFFFFF;
    --accent: #00d4ff;
}
```

### Add Screenshots
Replace the screenshot placeholders with actual game screenshots by adding image files and updating the paths in `index.html`.
